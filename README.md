## Codes par défaut

- Direction TACLAR : 0512
- Admin : 1111
- Dispatch : 2222
- Chauffeur : 3333

- bouton “Gestion des accès”
- modification des codes Direction / Admin / Dispatch / Chauffeur

- restauration des codes par défaut
- sauvegarde locale dans le navigateur
